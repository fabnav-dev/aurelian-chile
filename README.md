# Aurelian — El viaje por Chile

Carpeta lista para subir a cualquier hosting estático gratis.

## Opción 1 — Netlify Drop (más fácil, sin cuenta necesaria)
1. Ve a **https://app.netlify.com/drop**
2. Arrastra **toda esta carpeta** dentro del cuadro
3. Listo — te dan una URL pública (ej: https://aurelian-chile.netlify.app)
4. Manda esa URL por WhatsApp a la tía

## Opción 2 — Vercel
1. Ve a **https://vercel.com/new**
2. Sube esta carpeta
3. Listo

## Opción 3 — GitHub Pages
1. Crea un repo en GitHub
2. Sube los archivos
3. Settings → Pages → Branch: main → Save

## Archivos
- `index.html` — entrada principal
- `styles.css` — estilos
- `app.jsx` — código React
- `chapters.js` — contenido + traducciones ES/DE
- `photos/` — imágenes optimizadas
